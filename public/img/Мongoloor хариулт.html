CSS Modules-ийн үндсэн санаа

CSS Modules нь ямар ч анхааралгүйгээр нэг нэртэй CSS классуудыг (жишээ нь, .button) ашиглахыг зөвшөөрдөг. Гэхдээ бүх эдгээр нэр нь зөвхөн тухайн файлын хүрээнд байх бөгөөд global (глобал) бус, зөвхөн тухайн компонентын хүрээнд хязгаарлагдана. Үүний ачаар олон компонент зэрэг нэг нэртэй CSS класс ашигласан ч, тэд хоорондоо хөндлөнгөөс нөлөөлөхгүй.

1. CSS Module хэрхэн ашиглах вэ?

1.1. CSS файлыг модулиар импортлох

Эхлээд *.module.css өргөтгөлтэй CSS файлыг үүсгэнэ.

Button.module.css:

css
.button {
  background-color: blue;
  color: white;
  padding: 10px 20px;
}

1.2. React компонентод импорт хийх

Дараа нь энэ Button.module.css-г React компонентын кодонд импортлох хэрэгтэй.

Button.js:

javascript
import React from "react";
import styles from "./Button.module.css"; // Импортлох

const Button = () => {
  return (
    <button className={styles.button}>Click me</button> // `styles.button` ашиглан
  );
};

export default Button;

Энд styles.button нь Button.module.css доторх .button классыг зааж байна. styles объект нь CSS модулийн бүх классуудыг агуулсан байна. Энэ нь тухайн button классыг локаль (тодорхой компонент) болгож өгнө.

2. CSS Modules-ийн давуу талууд

2.1. Глобал нэр зөрчилгүй

  • Глобал CSS классуудыг ашиглахад ямар ч зөрчил үүсэх магадлалтай. Харин CSS Modules нь энэ асуудлыг шийдэж, тухайн компонентын хүрээнд нэрсийг scope (хүрээ) болгож өгдөг.

2.2. Голдуу локаль загварчлал

  • CSS модулийг ашиглах нь тухайн компонентын загварыг холбох, тусгаарлахад хялбар болгодог. Та зөвхөн тухайн компонентын загварыг засахад анхаарал хандуулж болно.

2.3. Автоматик нэрсийн өөрчлөлт

  • Систем нь тухайн элементүүдийн CSS классуудыг автоматаар нэрлэж, нэгээс олон компонент ашиглах үед нэр зөрчлөөс сэргийлдэг.

2.4. Засварлахад амар

  • CSS файлууд нь тусгаарлагдсан тул тухайн файлд хийгдсэн өөрчлөлт нь зөвхөн тухайн файлын компонентийг нөлөөлөх тул илүү хялбар засварлагдана.

2. { subsets: ["latin"], weight: "400" } параметрүүдийг тайлбарлая

🔹 subsets: ["latin"]

  • Фонтонд Latin үсгүүд хэрэгтэй бол "latin" гэж заана.

  • Хэрэв Монгол, Орос хэлний Кирилл үсгүүд хэрэгтэй бол "cyrillic" гэж нэмэх шаардлагатай.

javascript
const bebasNeue = Bebas_Neue({ subsets: ["latin", "cyrillic"], weight: "400" });

✅ Ингэснээр кирилл үсэгтэй текст зөв харагдана.

🔹 weight: "400"

  • Фонтын зузаан (font-weight)-г заана.

  • Bebas Neue фонт зөвхөн 400 жинтэй байдаг тул "400" гэж бичсэн байна.

  • Бусад фонтууд хэд хэдэн зузаантай байж болно (300, 400, 700 гэх мэт).

Жишээ нь:

javascript
const inter = Inter({ subsets: ["latin"], weight: ["400", "700"] });

✅ Inter фонтын хувьд 400, 700 зузааныг хоёуланг нь татна.

3. bebasNeue-г яагаад хадгалж байгаа вэ?

javascript
const bebasNeue = Bebas_Neue({ subsets: ["latin"], weight: "400" });

📌 Энэ хувьсагч (bebasNeue) нь дараа нь CSS-д ашиглагдана!

Жишээ нь, React компонентэд className болгож ашиглана:

javascript
<div className={bebasNeue.className}>
  Hello, this text uses Bebas Neue font.
</div>

✅ bebasNeue.className нь автоматаар зөв CSS-class үүсгэж, фонт хэрэгжүүлнэ.
